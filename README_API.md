#### 1.4  添加账单
>返回新增账单

**必选参数:**
* uiserId : 用户id
* sortid : 账单分类id
* payid ： 用户支付方式id
* cost ： 账单金额
* crdate ： 创建时间
* content ： 备注
* income ： 是否为收入

**接口地址:**
`bill/add`
